### 🤖 AI Student Productivity Tool
> *Search Less. Solve More.*

A full-featured AI-powered productivity platform built during a Hackathon —
designed to help students turn raw ideas into structured, actionable outcomes.

**What it does:**
- 🔍 **Deep Research** — Instantly researches any topic with AI-powered insights
- 🚀 **Project Planner** — Scaffolds complete project plans with structured phases
- ✍️ **Content Generator** — Creates posts for LinkedIn, Instagram & more
- 🎓 **Learning Path** — Builds personalized course roadmaps based on your goals
- 📋 **Project Management** — End-to-end AI-guided project execution

**Built with:** React.js • Claude AI • REST APIs • Tailwind CSS

> 💡 Idea → Research → Plan → Content → Learning Path → Output
> All in one place, all in seconds.
